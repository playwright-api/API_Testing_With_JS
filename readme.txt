'The API Testing with JS' is bult with Playwright: an Open Source tool by Microsoft. 
Please refer to the above link for for detailed installation steps to run the tests.

npm install 'If you do not have node.js package in your machine'
npm init -y
npm init playwright@latest 'This will install Playwright'
npx playwright test --ui 'to run the tests'
npx playwright show-report 'to check the report'
